# Thiago Social Media

Landing page estática para apresentação dos serviços de social media, design, conteúdo, vídeo, tráfego pago e landing pages.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Como conectar ao GitHub

Este projeto já está dentro de um repositório Git local e conectado ao GitHub. Para publicar, rode:

```powershell
git remote add origin https://github.com/tthiago-Marques/Landpagethiagosocialmedia.git
git add .
git commit -m "Cria landing page Thiago Social Media"
git branch -M main
git push -u origin main
```
